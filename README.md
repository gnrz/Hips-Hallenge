# Hips Hallenge

A browser-based tile puzzle game inspired by classic grid-based puzzle games. Navigate through levels collecting gems, using keys to unlock doors, and finding the exit.

## Play

Visit the [GitHub Pages site](https://gnrz.github.io/Hips-Hallenge/) to play in your browser.

## How to Play

## Levels

1. **Getting Started** - Learn the basics with a simple gem collection puzzle
2. **Key Hunt** - Navigate through obstacles to find keys and gems
3. **Locked Treasure** - A challenging puzzle with multiple gems, keys, and locked areas
